**PortfolioSite** — персональный сайт-портфолио VR/AR-разработчика.  
На сайте представлены ключевые проекты, краткая информация о разработчике и каталог остальных работ.

Проект создан как личная витрина для демонстрации опыта, навыков и завершённых проектов.

---

## 🎮 Основные разделы

- 🧑‍💻 **Главный экран** — краткое представление и специализация.
- 🧩 **Ключевые проекты** — избранные работы с описанием и ссылками.
- 📂 **Каталог проектов** — список остальных проектов.
- 🛠 **Навыки** — используемые инструменты и технологии.
- 📬 **Контакты** — ссылки для связи.

---

## 🧱 Стек

- **React 19**
- **Vite 7**
- **Tailwind CSS**
- **Framer Motion**
- **React Icons**
